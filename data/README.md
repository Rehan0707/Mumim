# data/ — Demo & seed assets  ·  Owner: Role 5 (with Role 3)

Curated assets that make the demo look alive. The backend seed script
(`backend/app/seed.py`) loads the catalog; screenshots/voice-notes drive the
Dikhao and voice demos.

| Folder | Contents |
| --- | --- |
| `seed-catalog/` | Product catalog (name, brand, price, attributes) + product photos |
| `screenshots/` | 2–3 Instagram-style screenshots that **do** match seed items (Dikhao) |
| `voice-notes/` | 1 clean Hindi/Marathi voice note for the voice-order demo |

Heavy media (mp4/wav/zip) is git-ignored; commit small images + CSVs.
